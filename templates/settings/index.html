Как работает планировщик

Система автоматически распределяет задачи по временным слотам на основе жадного алгоритма:

  1. Задачи сортируются по дедлайну и приоритету
  2. Для каждой задачи ищется ближайший свободный слот
  3. Если слот до дедлайна не найден — устанавливается флаг риска
  4. При завершении задач остальные перепланируются
